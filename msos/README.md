# Mr Sotong Operating System (MSOS)

MSOS is the operating system for Mr Sotong business operations, knowledge, workflows, dashboards, website, API, database design, and Notion-backed source of truth.

This repository is currently in the design and governance phase. It contains no production application code.

## Repository Structure

```text
msos/
├── docs/
├── prompts/
├── workflows/
├── dashboard/
├── website/
├── api/
├── database/
├── notion/
└── changelog/
```

## Operating Rules

- Never modify production directly.
- Design before coding.
- Test before deployment.
- Every change must include a version, changelog, and rollback plan.
- Follow the MSOS Constitution.
- Notion is the single source of truth.
- Reuse existing components whenever possible.
- Prefer simplicity over unnecessary complexity.
- Document every important decision.
- Ask before implementing when requirements are unclear.
